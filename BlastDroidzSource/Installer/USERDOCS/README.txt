TeamA001: version 1.0

Notes: Hello, we are TeamA001 and we would like to welcome you to our game!
Our game is best experienced using a XBOX 360 controller but we also have
some basic keyboard controls at the moment. After the game ends there is a
survey we would appreciate if you fill out with any comments or suggestions
you would like to critique and add to our game. Thank you for playing!

Controls:
  Keyboard:
    Movement: WASD controls
    Shoot   : Spacebar
    V       : Spawn more enemies
    Quit    : Escape

  XBOX 360 Controller:
    Movement   : Left Thumbstick
    Propulsion : Right Trigger
    Shoot      : A
